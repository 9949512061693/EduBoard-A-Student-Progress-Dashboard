# EduBoard – Student Progress Dashboard

EduBoard is a responsive student dashboard interface designed to simulate a modern educational portal where students can monitor their academic performance and important academic information in one place. The project focuses on building a clean and structured user interface using fundamental front-end technologies while maintaining usability and responsive design principles.

The dashboard provides an overview of key student metrics such as GPA, attendance percentage, assignments due, and total academic credits. It also includes a subject progress tracking table, upcoming assignment deadlines, and a student profile section displaying academic details. The goal of the project is to organize student information in a visually clear and accessible dashboard layout similar to those used in educational management systems.

The interface is structured into three primary sections: a sidebar navigation menu for quick access to different dashboard areas, a top header containing notification and student profile elements, and a main content area that presents academic data using cards, tables, and structured components.

To improve usability and visual clarity, the dashboard uses card-based layouts for displaying key statistics, progress bars for subject completion, and badge elements to highlight upcoming deadlines. Font Awesome icons are integrated to provide intuitive visual indicators throughout the interface.

The layout is fully responsive and adapts to different screen sizes, including desktop, tablet, and mobile devices. Bootstrap’s grid system and modern layout techniques are used to maintain consistent spacing, alignment, and responsiveness across the entire interface.

This project helped strengthen my understanding of front-end development concepts such as semantic HTML structuring, responsive design techniques, layout management using Bootstrap Grid and Flexbox, and designing user-friendly dashboard interfaces.

## Features

* Responsive dashboard layout
* Sidebar navigation menu
* Student overview cards (GPA, Attendance, Assignments Due, Credits)
* Subject progress table with progress indicators
* Upcoming deadlines section
* Student profile information card
* Clean and modern UI design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome

## Project Purpose

The purpose of this project is to practice building responsive front-end dashboard interfaces and to demonstrate the ability to structure and present information clearly using modern web development techniques.
